# Binary Clock — Design

## Default Theme
Warm Sunset (ID: 0)

## Color Role Mapping

| Role | UI Element |
|------|-----------|
| `primary` | (unused) |
| `accent` | Digital time text below grid |
| `secondary` | Date text at top |
| `highlight` | Active (on) dots, colon separators |
| `muted` | Off dot stroke, dim invalid dots, row labels, column labels |

## Notes
- Replaces the old day/night PictonBlue/Green theme system with the shared pastel design system
- The old `s_is_night` flag and time-of-day color switching have been removed
- Battery indicator is not present in this watchface
