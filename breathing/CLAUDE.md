# Breathe

Guided breathing exercise with visual and haptic feedback.

## How It Works
- Expanding/contracting circle guides inhale/exhale
- 4-7-8 breathing pattern: inhale 4s, hold 7s, exhale 8s
- Gentle vibration at each phase transition
- Select: Start/Stop session
- Up/Down: Switch between patterns (4-7-8, Box, Simple)
- Cycle counter shows completed breaths

## Patterns
- **4-7-8**: Inhale 4s, Hold 7s, Exhale 8s (relaxation)
- **Box**: 4s each — Inhale, Hold, Exhale, Hold (focus)
- **Simple**: Inhale 4s, Exhale 4s (basic)

## Platforms
- diorite, basalt, emery
