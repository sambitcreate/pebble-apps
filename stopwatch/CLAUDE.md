# Stopwatch

Precision stopwatch with lap time tracking.

## How It Works
- Large MM:SS.t display (tenths of seconds)
- Select: Start/Stop
- Up: Lap (records split time while running)
- Down: Reset (when stopped)
- Shows last 3 lap times on screen
- Vibration on lap

## Platforms
- diorite, basalt, emery

## Build
```bash
pebble build
pebble install --emulator diorite
```
