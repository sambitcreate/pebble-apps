# Pixel Invaders

Classic space invaders-style shooter on your Pebble.

## How It Works
- 8 columns x 4 rows of aliens march across and down the screen
- Player ship at bottom moves left/right
- Up: Move left
- Down: Move right
- Select: Fire bullet
- Aliens speed up as you destroy them
- Score and lives displayed
- Vibration on hit and death

## Platforms
- diorite, basalt, emery

## Build
```bash
pebble build
pebble install --emulator diorite
```
