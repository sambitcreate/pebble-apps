# Reaction Tester

Test your reaction time on a Pebble watch.

## How It Works
- Press Select to start a round
- Wait for the screen to flash and vibrate (random 1-5 second delay)
- Press Select as fast as you can
- See your reaction time in milliseconds
- Tracks best time across rounds
- Press Up to see your best, Down to reset

## Platforms
- diorite, basalt, emery

## Build
```bash
pebble build
pebble install --emulator diorite
```
