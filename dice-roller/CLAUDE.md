# Dice Roller

Shake or tap to roll dice. Perfect for board games and D&D.

## How It Works
- Shake wrist or press Select to roll
- Up/Down: cycle die type (D4, D6, D8, D10, D12, D20)
- Visual die face for D6, numeric for others
- Shows roll history (last 5 rolls)
- Vibration feedback on roll

## Platforms
- diorite, basalt, emery

## Build
```bash
pebble build
pebble install --emulator diorite
```
