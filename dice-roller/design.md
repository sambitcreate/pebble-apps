# Dice Roller - Pastel Theme Color Mapping

| UI Element          | Previous Color               | Theme Role           |
|---------------------|------------------------------|----------------------|
| Status bar text     | White                        | `s_theme.primary`    |
| Die type label      | Yellow / White               | `s_theme.accent`     |
| D6 face fill        | White                        | Unchanged (White)    |
| D6 pips             | Black                        | Unchanged (Black)    |
| D6 outline stroke   | DarkGray / Black             | `s_theme.muted`      |
| Non-D6 polygon fill | DarkGray / White             | `s_theme.muted`      |
| Non-D6 number text  | White / Black                | `s_theme.primary`    |
| Combo streak text   | Red / White                  | `s_theme.highlight`  |
| Result text         | White                        | `s_theme.primary`    |
| Stats text          | Green / White                | `s_theme.accent`     |
| History text        | DarkGray / LightGray         | `s_theme.muted`      |
| Overlay text        | White                        | `s_theme.primary`    |

## Theme Selection

Configurable via phone settings (Settings > Dice Roller). Default: Warm Sunset.

## Notes

- D6 face remains white with black pips for readability.
- Theme picker is added alongside existing Die Type and Show Stats options.
