# Magic 8-Ball

Shake your wrist to reveal the future. Classic Magic 8-Ball for Pebble.

## How It Works
- Shake the watch (accelerometer tap detection) to get a random answer
- 20 classic Magic 8-Ball responses
- Visual triangle with answer text
- Vibration feedback on shake
- Press Select to ask again manually

## Platforms
- diorite, basalt, emery

## Build
```bash
pebble build
pebble install --emulator diorite
```
